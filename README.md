# AstrBot 算卦插件 🔮

一个基于易经六十四卦的 [AstrBot](https://astrbot.app) 聊天机器人插件，通过 AI 函数工具调用实现智能算卦解卦。

## 免责声明

插件开发全程均由 AI 完成

## 功能特点

- 🤖 **AI 函数工具调用**：作为 LLM 工具供 AI 调用，实现智能算卦
- 🎲 **随机起卦**：随机生成六十四卦中的某一卦
- 📖 **完整卦象信息**：包含卦名、卦象、性质、含义、爻辞、运势指引
- 💬 **自然对话**：用户无需记忆命令，直接与 AI 对话即可触发

## 安装方式

### 方法一：通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入插件市场
3. 搜索关键词 "算卦" 或 "suanua"
4. 安装

### 方法二：手动安装

```bash
# 在 AstrBot 的 data/plugins 目录下
git clone https://github.com/86lbs/astrbot_plugin_suanua.git
```

然后重启 AstrBot。

## 使用方法

### 与 AI 对话触发

用户无需记忆任何命令，直接与 AI 对话即可：

```
用户: 帮我算一卦
AI: 好的，让我为您起一卦...

【乾卦】
━━━
━━━
━━━
━━━
━━━
━━━
卦性：纯阳
含义：天行健，君子以自强不息。
爻辞：九三：君子终日乾乾，夕惕若，厉无咎。
运势指引：
  • 事业方面：脚踏实地，稳扎稳打。
  • 感情方面：真诚待人，缘分自来。
  • 财运方面：量入为出，积少成多。

根据乾卦的卦象，我来为您详细解卦...
```

### 支持的对话方式

- "帮我算一卦"
- "我想知道我的事业运势"
- "帮我占卜一下感情"
- "今天运势怎么样"
- "算一卦问问工作"

AI 会自动识别用户意图并调用算卦工具。

## 输出内容

每次算卦会输出以下信息：

| 内容 | 说明 |
|------|------|
| 卦名 | 六十四卦之一 |
| 卦象 | 六爻图形显示 |
| 卦性 | 卦的基本性质 |
| 含义 | 卦的基本含义 |
| 爻辞 | 随机选取的一爻爻辞 |
| 运势指引 | 事业、感情、财运、健康等方面的建议 |

## 技术实现

- **LLM 工具**：使用 `@llm_tool` 装饰器注册为 AI 可调用的函数工具
- **随机起卦**：从六十四卦中随机选择一卦
- **本地数据**：包含完整的六十四卦 JSON 数据

## 注意事项

1. 本插件仅供娱乐参考，请勿迷信
2. AI 解卦结果由大语言模型生成，仅供参考
